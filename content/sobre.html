Sobre o projeto

Este é um projeto do grupo de pesquisa Formação docente e tecnologias de informação e comunicação (veja lista completa de participantes no CNPq) que desde 2011 estuda a Universidade Aberta do Brasil (UAB). O grupo é coordenado pela UFF com parceria da UEL, UNICAMP. Este mapeamento foi conduzido pela Cátedra UNESCO em Educação Aberta na UNICAMP.
O projeto

O mapa apresenta duas bases de dados:

Primeiro, uma base de informações com dados coletados através de pedidos via Lei de Acesso a Informação e outros extraídos do sistema SisUAB em março de 2016. Trazemos a tona um mapa com todas as IPES (Instutições Públicas de Ensino Superior) e todos os polos ativos, que juntos, compõe o Sistema UAB. Os dados de geolocalização são aproximados. Buscamos apontar com a maior especificidade possível, a localização das IPES (feitas manualmente). Os polos forma geolocalizados automaticamente e referem a cidade, sem precisão no nível de logradouro.

Segundo informações detalhadas sobre os Núcleos de Educação a Distância de 8 universidades que participaram de uma pesquisa realizada pelo nosso grupo focada na gestão de recursos educacionais no âmbito da UAB. Para essas instituições incluímos um texto introdutório e uma detalhada linha do tempo de sua participação na UAB. Esses espaços são responsáveis pela articulação, dentro da universidade, para a efetivação dos cursos oferecidos pela UAB. Como cada universidade tem autonomia para definir seu modelo de ação, cada um desses núcleos se estrutura de maneira diferente. O objetivo desses estudo é descrever e avaliar questões relativas ao funcionamento e estrutura desses centros, com particular atenção aos processos de produção e disseminação de recursos educacionais.

Caso identifique algum problema, dado incongruente, ou queira ajudar a melhorar o sistema, entre em contato colocando um problema/comentário na nossa página no Github.

O software

O sistema que desenvolvemos, apelidado de Ciclope, é um software livre, que agrega a funcionalidade de diversas ferramentas Existem duas versões do sistema. A primeira não depende de banco de dados e faz uso inteiramente de dados disponíveis no formato JSON usando arquivos-texto para armazenamento dos dados. O sistema faz uso de HTML + Javascript (Ciclope). Com base nesse primeiro, desenvolvemos o CiclopeDB, que utilizamos nesse site. O software faz uso de um banco de dados SQL + Javascript + HTML + PHP. É possível verificar e adicionar problemas, questões e sugestões sobre o software no site do CiclopeDB no Github. Ambos estão licenciadas com a licença GPL v3.0.
